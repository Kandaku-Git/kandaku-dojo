Origines du karaté Shotokan

Le karaté Shotokan trouve ses racines dans les arts martiaux d’Okinawa, influencés par différentes écoles de combat asiatiques.

L’objectif de cette discipline est de développer un corps fort, un esprit clair et un comportement respectueux, au-delà de la simple dimension sportive.

Histoire du dojo virtuel

Ce dojo virtuel a été conçu comme un support d’étude pour les pratiquants souhaitant réviser en autonomie, en complément de l’enseignement en salle.

L’objectif est de proposer une interface claire, rapide et personnalisable pour retrouver l’essentiel des techniques et ressources.

Tutoriel d’utilisation de l’interface

Le site fonctionne comme une petite application : tu utilises le bouton MENU pour ouvrir la navigation, choisir une section, puis le menu se replie automatiquement.

  1. Clique sur le bouton MENU pour afficher le menu.
  2. Choisis une rubrique (Techniques, Vidéos, Lexique…).
  3. Pour les techniques, ouvre une catégorie puis clique sur une technique pour lancer le diaporama.
  4. Dans Vidéos, clique sur une vignette pour ouvrir la vidéo dans un nouvel onglet.
  5. Dans Lexique, tape un mot-clé pour filtrer les termes en temps réel.
  6. Dans Personnalisation, modifie les couleurs pour adapter l’ambiance à ton goût.
